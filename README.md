# VinculoHubPortal

## 🚀 Como rodar o projeto
### Pre-requisitos: 
- Java 17 ou superior
- Node.js 18 ou superior

### ☕ Backend (Spring Boot)
1. Navegue até a pasta do backend:
2. `cd backend`
3. Execute o projeto usando o Maven Wrapper:
   - MAC e Linux: `./mvnw spring-boot:run`
   - Windows: `mvnw spring-boot:run`
   
   Nota: O projeto iniciará com um erro de DataSource até que o banco de dados seja configurado. Isso é esperado.

### ⚛️ Frontend (React + Vite)
1. Navegue até a pasta do frontend:
2. cd frontend
3. Instale as dependências rodando npm install
4. Inicie o servidor de desenvolvimento rodando npm run dev
5. Acesse: http://localhost:5173/
